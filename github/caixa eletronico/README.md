  # Sistema de Caixa Eletrônico com Histórico de Extrato ??

 Este é um projeto desenvolvido em Portugol através do ambiente **VisuAlg**,
simulando as operaçőes essenciais de um terminal de autoatendimento bancário (Caixa Eletrônico).
 O sistema conta com validaçőes defensivas de dados e armazenamento de transaçőes em tempo real.

> ?? **Status do Projeto:** Em desenvolvimento ??

---

## ?? Funcionalidades do Sistema

- [x] **Cadastro de Correntistas:** Módulo de cadastro limitado a 5 usuários com
validaçăo inteligente de strings(rejeita números e caracteres especiais).
- [ ] **Módulo de Login:** Autenticaçăo de segurança para acesso ŕs contas.
- [ ] **Depósitos Bancários:** Entrada de valores com atualizaçăo imediata do saldo.
- [ ] **Saques Eletrônicos:** Retirada de valores com validaçăo física de saldo disponível.
- [ ] **Histórico de Extrato:** Vetor dinâmico que armazena a descriçăo e
os valores das últimas operaçőes realizadas pelo usuário.
